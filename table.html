Untuk membuat tabel dalam HTML, pertama-tama kamu harus menggunakan tag table sebagai pembungkus utama tabel. Di dalamnya, kamu bisa menambahkan tag thead untuk bagian kepala tabel yang biasanya berisi judul kolom, dan tag tbody untuk data tabelnya. Setiap baris dalam tabel didefinisikan dengan tag tr. Di dalam tag tr, kamu bisa menggunakan tag th untuk mendefinisikan judul kolom di kepala tabel, dan tag td untuk mengisi sel data di bagian isi tabel.
