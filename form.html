Login

Pertama-tama kita menggunakan tag form sebagai wadah untuk elemen-elemen form. Di dalam tag ini, kita tambahkan dua kolom input, satu untuk username menggunakan input type="text" dan satu untuk password menggunakan input type="password". Masing-masing input biasanya dilengkapi dengan label menggunakan tag label untuk memberikan keterangan bagi pengguna. Setelah itu, tambahkan tombol kirim atau login menggunakan button atau yang berfungsi untuk mengirim data yang telah diisi. Form ini akan mengirimkan data ke URL
